# **App Name**: ShadowScribe

## Core Features:

- Entry Input: Allows the user to type their journal entry into a digital notebook interface.
- AI Shadow Response: Generates a short, punchy, psychoanalytic response from the perspective of the user's 'shadow self,' based on the user's entry, observing contradictions using the persona of a detached, hyper-observant clinician, leveraging a tool for reasoning and incorporation of insights from the user's text.
- Page Turn Transition: Implements a slow, heavy page turn animation triggered by a swipe gesture to transition between user input and AI response.
- Shadow Text Reveal: Displays the AI-generated text letter-by-letter, simulating handwriting, on a darker, grittier page.
- Emotional Tagging: Analyzes each entry to identify and store 'emotional tags' such as Avoidance, Narcissism, and Guilt, utilizing vector database.
- Monthly Psychological Report: Generates a monthly psychological profile summarizing the user's patterns and emotional trends based on the stored emotional tags. Creates and saves psychological profiles based on recent journal history to the 'other' database
- Patient File UI: Presents the monthly psychological report as a manila folder appearing in the UI on the 1st of each month.

## Style Guidelines:

- Primary color: Dark, muted grey (#4A4A4A) to evoke a sense of depth and introspection. Background color: Very dark grey (#1E1E1E). Accent color: Muted, desaturated blue (#708090).
- User font: 'Literata', serif, for journal entry input. Shadow font: 'Space Grotesk', sans-serif, for AI response, evoking a different tone.
- Minimalist icons for UI elements, focusing on clarity and ease of use.
- Skeuomorphic design mimicking a worn black leather notebook. Dark mode by default with subtle textures to enhance the tactile feel.
- Slow and heavy page turn animation with haptic feedback. Letter-by-letter text reveal for AI response with accompanying heavy, low-frequency thud sound.
